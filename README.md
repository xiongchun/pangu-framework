### 介绍

[**盘古开发框架**](https://pulanos.gitee.io/pangu-framework) 是一套轻量稳健的工业级前、中、后台三维多端行业数字化赋能开发基座。基于商业友好的 [Apache-2.0](https://www.apache.org/licenses/LICENSE-2.0) 协议 **免费开源** 发布。包含:  
- [盘古服务开发框架](https://pulanos.gitee.io/pangu-framework/docs/intro)
- [盘古中后台业务系统开发脚手架](https://pulanos.gitee.io/pangu-framework/admin/intro) 
- [盘古移动端多平台开发脚手架](https://pulanos.gitee.io/pangu-framework/app/intro)
